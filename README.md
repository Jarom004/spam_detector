# Practica Spam
Esta es una practica de un detector de spam.

## De que trata la practica 
En esta practica se entrenó el detector con datos ham y datos spam, con este entrenamiento detecta e indica si el texto o email es spam o no es spam.

## Como?
1- Creamos la carpeta spam_detector
2- En un entorno virtual descargamos las librerias necesarias
3- Creamos 3 archivos el principal, el de la ventana emergente y el detector de spam
4- Subimos un csv con todos los ham y spam para entrenar
